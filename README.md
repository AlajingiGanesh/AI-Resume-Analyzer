# AI Resume Analyzer

AI Resume Analyzer is a full-stack web application that helps job seekers analyze, improve, and tailor their resumes using AI.

It simulates how Applicant Tracking Systems (ATS) and recruiters evaluate resumes against job descriptions.

## Features

- Resume PDF upload and analysis
- AI-powered resume rewriting
- Skill gap roadmap generation
- Mock interview with scoring
- Career chatbot assistant
- History tracking
- PDF export of results

## Tech Stack

### Backend
- FastAPI (Python)
- Groq / LLM APIs
- PDF parsing

### Frontend
- HTML, CSS, JavaScript
- Glassmorphism UI
